# 新型コロナウイルス感染症 支援情報ナビ オープンソース版

「新型コロナウイルス感染症 支援情報ナビ オープンソース版」は、行政サービス情報の詳細データを読み込ませることで簡単に支援情報ナビを作成できるプロジェクトです。東京都の「新型コロナウイルス感染症 支援情報ナビ」と同様の「自分にあった制度を探す」、「テーマ別に制度を見る」の機能が使用できます。また、行政サービス情報やナビゲーションロジックのファイルを用意するだけで、簡単にデータの更新ができます。

## 機能
- 東京都の「新型コロナウイルス感染症 支援情報ナビ」と同様の「自分にあった制度を探す」、「テーマ別に制度を見る」の機能
- 行政サービス情報ファイルの準備で簡単にデータ更新が可能
- ナビゲーションロジックファイルの準備で簡単にロジック更新が可能

## 必要環境
- Node.js
- npm

## 使い方
1. リポジトリをクローンします。
2. 依存モジュールをインストールします。
3. `npm run build` を実行してサイトファイルを生成します。
4. 行政サービス情報とナビゲーションロジックのCSVファイルを準備して、ファイルをアップロードします。
5. ローカルサーバーを立ち上げて、ブラウザでサイトを確認します。

詳細な利用方法については、リポジトリ内の `manual.pdf` をご確認ください。
